- ใช้ API: [PokéAPI](https://pokeapi.co/)
- แสดงผล:
  - ช่องทีมด้านบน 3 ช่อง (แสดงตัวที่เลือก)
  - รายการ Pokémon ทั้งหมดให้เลือก
  - Search ค้นหาได้
  - ปุ่ม Create Team (กดได้ก็ต่อเมื่อเลือกครบ 3)
  - หน้า My Teams แสดงทีมที่บันทึก
